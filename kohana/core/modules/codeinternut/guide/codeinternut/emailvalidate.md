# Validador de E-mails

Esse método consiste em verificar se um e-mail é válido ou não.

Abaixo um breve resumo de como utilizar esse modulo.

## Uso
Para instanciar essa função utilize dentro de sua aplicação o código abaixo.
~~~~
Codeinternut::emailvalidate($email);
~~~~

### $email
Ao instanciar essa função informe em formato string o e-mail a ser válido esse método retorna true ou false;
